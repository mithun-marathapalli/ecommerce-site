This is a ecommerce website built with [Next.js](https://nextjs.org) and deployed on [Vercel](https://ecommerce-site-alpha-mocha.vercel.app/).

## Getting Started

Recommended Node version: 22.x

To get a local copy up and running, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/mithun-marathapalli/ecommerce-site.git
cd ecommerce-site
```

2. Install dependencies:

```bash
npm install
```

or

```bash
yarn install
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

-   Product listing and details
-   Shopping cart functionality

## Deployment

The site is deployed on Vercel and you can visit it [here](https://ecommerce-site-alpha-mocha.vercel.app/).
